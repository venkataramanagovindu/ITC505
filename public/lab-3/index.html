Delivering a Custom Font to a Client

Font Analysis: Lobster

The "Lobster" font is known for its distinctive, playful, and somewhat vintage look. Here are some characteristics and analysis of this font:

  • Design Style: Lobster has a bold, cursive style with elaborate letterforms that convey a sense of fun and elegance. Its thick strokes and rounded edges give it a friendly and approachable feel.
  • Usage: Due to its unique style, Lobster is often used in branding, logos, posters, and titles where a distinctive and eye-catching font is desired. It is particularly popular in retro and vintage designs.
  • Legibility: While Lobster is readable, its decorative nature can make it harder to read in long passages of text. It is best used for headings, short phrases, or decorative purposes.
  • Popularity: Lobster has become quite popular in web design and graphic design due to its distinctive look and versatility in creating engaging and memorable designs.

Overall, Lobster is a font with a strong visual identity that can be highly effective when used appropriately. Designers should be mindful of its potential to dominate the visual hierarchy and ensure it is used in contexts where its style enhances the overall design.

This page demonstrates how to deliver a custom font to a client's browser using the CSS @font-face rule. The steps involved are as follows:

  1. Choose a Font: Choose a font-style and select a font file (e.g., .ttf, .woff, .woff2). For this example, we chose the "Lobster" font.
  2. Host the Font File: Upload the font file to your server. In this case, the font file is hosted at /fonts/Lobster-Regular.ttf on the server. (in /lab-3 directory)
  3. CSS @font-face Rule: Define the font in your CSS using the @font-face rule, specifying the path to the font file on your server. Here is the CSS rule used: 
    
@font-face {
    font-family: 'Lobster';
    src: url('fonts/Lobster-Regular.ttf') format('truetype');
}
body {
    font-family: 'Lobster', sans-serif;
}
  4. Apply the Font: Use the defined font family in your CSS styles to apply it to elements on your webpage. For this example, the body element is styled with the "Lobster" font: 
    
body {
    font-family: 'Lobster', sans-serif;
}

The result is this page, which uses the "Lobster" font for its text. This method ensures that even if the font is not installed on the client's computer, it will still be displayed correctly.
